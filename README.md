## Tech Stack

| Layer           | Technology                          |
|------------------|--------------------------------------|
| Backend          | Spring Boot 3.4.4                    |
| View Layer       | JSP, JSTL                            |
| Security         | Spring Security                      |
| Build Tool       | Maven                                |
| Database         | MySQL (Dockerized)                   |
| ORM              | Spring Data JPA                      |
| Frontend Styling | Bootstrap (via WebJars), jQuery      |
| Validation       | Spring Boot Validation               |
| Dev Tools        | Spring Boot DevTools (Live Reload)   |
